This code repo is for Coding Challenges - Build Your Own JSON Parser
URL: https://codingchallenges.fyi/challenges/challenge-json-parser

Answer below questions before jumping to coding:
1. What language am I supporting?
   (Grammar)

2. What information must survive tokenization?
   (Token types)

3. How do I recognize those tokens character by character?
   (Lexer states and transitions)

4. How do those tokens combine into valid constructs?
   (Parser states / recursive-descent functions)
